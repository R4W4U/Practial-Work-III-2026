# Microsoft Defender for Endpoint Power BI Dashboard

## Overview
This project focuses on the development of a Power BI dashboard for Microsoft Defender for Endpoint (MDE). Data is retrieved from MDE using Kusto Query Language (KQL) queries executed through the Microsoft Graph API and visualized in Power BI.

The goal is to create a modular and customer-oriented dashboard that provides better endpoint security visibility than the standard MDE dashboards.

## Objectives
- Retrieve security data from Microsoft Defender for Endpoint.
- Connect MDE data to Power BI using the Microsoft Graph API.
- Develop reusable dashboard components based on best practices and customer requirements.
- Provide clear and actionable endpoint security insights.
- Improve knowledge and practical experience in Power BI, KQL, and Microsoft Defender technologies.

## Technology Stack
- Microsoft Defender for Endpoint
- Microsoft Graph API
- Kusto Query Language (KQL)
- Microsoft Power BI
- DAX & Power Query

## Project Scope
1. Research Microsoft Defender for Endpoint, KQL, Graph API, and Power BI.
2. Develop KQL queries to collect relevant security data.
3. Integrate Defender data into Power BI.
4. Design and implement modular dashboard components.
5. Test and validate data quality and dashboard functionality.
6. Optionally deploy and validate the solution in a customer environment.

## Expected Outcome
A documented Power BI dashboard solution that enables customers to visualize important endpoint security information using customizable dashboard modules based on best practices and business requirements.
